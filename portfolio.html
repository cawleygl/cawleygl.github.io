A simple React front-end application for the randomuser.me API that allows users to organize and view information about the members of a team of employees. Weather Dashboard
A simple weather dashboard that shows weather conditions in multiple cities using the OpenWeather API. Written in HTML, CSS, and Javascript also using the Materialize CSS framework. Also saves previously searched cities using local storage. Type a city name into the search bar and click submit to find the weather in that city!

Workday Planner
A HTML / Bootstrap CSS webpage that uses Javascript, Moment.js, and local storage to schedule out events in a work day. Users can type events or other reminders into the blocks on the page and save them by clicking on the save buttons on the right of each block. As the day progresses, the page formatting will highlight the current block and strike events in past blocks.
